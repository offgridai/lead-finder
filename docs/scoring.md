# Scoring Model Notes

This project uses evidence-weighted estimates.

## Activity score

Initial placeholder signals:

- online booking present
- recent upgrades or expansion language
- review trend
- website freshness
- social activity
- permit or amenity changes

## Cashflow estimate

Initial placeholder model:

```text
annual gross revenue = unit_count × nightly_rate × occupied_nights
cashflow = gross revenue × cashflow margin
```

Use ranges, not point estimates.

Future calibration should use known operator P&Ls from trusted sources.

## Transition likelihood

Initial placeholder signals:

- long-term ownership
- family-operated branding
- retirement or succession language
- stale digital presence
- lack of reinvestment
- recent reinvestment reducing likelihood
- second-generation management reducing likelihood

Use respectful language: "openness to acquisition discussion," not predatory language.
